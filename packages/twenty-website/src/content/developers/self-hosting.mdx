Self-hosting Twenty lets you run the app on your own infrastructure. The quickest way is to use the Docker images provided in this repository.

## Quick start

1. Copy the example environment file and create your own `.env`:
2. Edit `.env` to set secrets such as `APP_SECRET` and your database credentials.
3. Start the containers:

```bash
docker compose up -d
```

Once running, open [http://localhost:3000](http://localhost:3000) or the URL defined in `SERVER_URL`.

Detailed instructions are available in the [Docker Compose guide](https://twenty.com/developers/section/self-hosting/docker-compose).
